# MacOS Website Blocker GUI

This is Simple Website Blocker With Simple and Clean GUI.

## How do I use it?

Run below script to install PySimpleGUI library:

     pip install PySimpleGUI

To run the script:

    python blocker.py


## What's it doing?

The script is writing hosts file in /private/etc/. Due to which in return you will not able to access that particular website.
